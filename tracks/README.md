# Tracks

Drop `.mp3` / `.wav` / `.ogg` files here.

Two ways the page finds them:
1. **Served over http** — list filenames in `tracks.json`, e.g. `["song1.mp3","song2.mp3"]`. They auto-load.
2. **Anywhere (even file://)** — just drag files onto the page, or use “＋ Add audio files”.

`test-tone.wav` is a generated rhythmic clip for testing the visualizer.
